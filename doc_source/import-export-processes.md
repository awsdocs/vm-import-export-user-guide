# VM Import/Export Processes<a name="import-export-processes"></a>

VM Import/Export has processes for eligible resources that you can use to import into and export out of the AWS Cloud\. You can import individual disks, or whole VMs that meet the respective requirements of the import process\. You can also export an Amazon EC2 instance or an AMI in a supported file format\. For more information on resources that are eligible for export, see *[Considerations for instance export](vmexport.md#vmexport-limits)* and *[Considerations for image export](vmexport_image.md#limits-image-export)*\.
